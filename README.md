# ancrage.io

Static site for Ancrage — an embedded operating partner with a technical edge.

## Pages

- `/` — Landing page
- `/contact` — Get in touch
- `/privacy` — Privacy policy
- `/terms` — Terms of service

## Deploy

Hosted on Vercel. Push to `main` to deploy.

## Stack

- Static HTML/CSS
- Plus Jakarta Sans (Google Fonts)
- Vercel for hosting with `cleanUrls` enabled
- No build step required

## Domain

Custom domain: `ancrage.io` — configure in Vercel dashboard under Settings → Domains.
